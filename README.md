# Moyenne Mediane

Moyenne Mediane, comme le nom l'indique, affiche la moyenne et la mediane d'un ensemble de nombres.

## Installation

Installer Moyenne Mediane

```bash
yarn install
```

## Lancer l'application

Démarrer le projet

```bash
npm run dev
```

Générer les styles avec Tailwind + watch

```bash
npm run tww
```
